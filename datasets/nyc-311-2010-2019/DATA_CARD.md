# NYC 311 2010-2019

This directory contains a four-row fixture for Socrata dataset `76ig-c548`. The full contract uses an explicit column list, a 2010-01-01 through 2020-01-01 time bound, deterministic `unique_key` ordering, and paged responses. The live source is mutable and has no machine-readable license declaration in the portal metadata.

Source: [NYC Open Data API](https://data.cityofnewyork.us/resource/76ig-c548.csv). Retain the NYC [terms](https://opendata.cityofnewyork.us/overview/) and raw response pages before publishing a full snapshot.
