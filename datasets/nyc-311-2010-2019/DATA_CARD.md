# NYC 311 2010-2019

This directory contains a four-row fixture and a one-million-row normalized benchmark snapshot for Socrata dataset `76ig-c548`. The snapshot uses an explicit column list, a 2010-01-01 through 2020-01-01 time bound, and deterministic `unique_key` ordering. The full CSV is distributed as a checksum-pinned Release asset. The live source is mutable and has no machine-readable license declaration in the portal metadata.

The current `v0.2.0-rc1` asset predates raw-page retention. Its immutable CSV hashes remain verifiable, but its missing raw pages cannot be reconstructed honestly from the mutable endpoint. Do not treat a later live capture as the source evidence for that asset. Future captures use manifest-driven keyset pagination on Socrata's unique internal `:id`; offset pagination is forbidden. They retain raw CSV pages and metadata, record page hashes and cursor ranges, and fail if `rowsUpdatedAt` changes. A changed capture requires a new immutable dataset identity and Release asset. Run `uv run --frozen format-bench dataset capture nyc-311-2010-2019 --output .data/captures/<new-capture-id>`; the command is append-only. Primary sources: [NYC API](https://data.cityofnewyork.us/resource/76ig-c548.csv), [metadata](https://data.cityofnewyork.us/api/views/76ig-c548), [system fields](https://dev.socrata.com/docs/system-fields.html), [ordering](https://dev.socrata.com/docs/queries/order), and [filtering](https://dev.socrata.com/docs/queries/where.html). Retain the NYC [terms](https://opendata.cityofnewyork.us/overview/).
