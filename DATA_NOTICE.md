# Data Notice

The Apache-2.0 license covers this repository's code and original documentation. It does not relicense third-party repository names, descriptions, topics, URLs, or other GitHub-hosted content.

The `github-stars-2026-07-03` case study contains public repository metadata collected for open-access research. Its source and known limitations are recorded in the Data Card and manifest.

- GitHub Starring API: https://docs.github.com/en/rest/activity/starring
- GitHub Acceptable Use Policies: https://docs.github.com/en/site-policy/acceptable-use-policies/github-acceptable-use-policies
- Source profile: https://github.com/steipete?tab=stars

Do not use the dataset for spam, identity inference, recruiting lists, or decisions about individuals. Never commit GitHub credentials or private repository data.

For correction or removal requests, open an issue in `Anionix/data-format-lab` and identify the affected repository row.
